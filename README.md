# Tablekard - QR Ordering for Restaurants

Tablekard is a QR-code ordering system for restaurants. We onboard your team, digitize your menu, and launch a table-first ordering experience so guests can scan, order, and pay without waiting.

## Highlights

- Clean, modern landing page inspired by lightweight SaaS layouts
- Phone mockups that visualize the guest ordering flow
- Sections for About, Features, and Blogs
- Dedicated legal pages for Terms, Privacy, and Refunds

## Project Structure

- index.html
- style.css
- script.js
- privacy.html
- terms.html
- refund.html

## Run Locally

Open index.html directly in your browser or use a local server.
