# Luxe Dine - Premium In-House QR Ordering System

![Luxe Dine Banner](https://images.unsplash.com/photo-1559339352-11d035aa65de?auto=format&fit=crop&q=80&w=1548)

Luxe Dine is a high-end, premium landing page for an in-house QR ordering system designed for a luxurious dining experience. It streamlines the ordering process by allowing customers to scan, browse, and pay directly from their table.
